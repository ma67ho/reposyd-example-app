# Quickstart for Developers

1. `yarn install`
2. Set up the [RePoSyD service](https://github.com/ma67ho/reposyd-example-service/blob/main/doc/quickstart-dev.md) first.
3. `yarn global add @quasar/cli`
4. `quasar dev` (If this fails, make sure to have the [global yarn bin](https://classic.yarnpkg.com/en/docs/cli/global) folder in your PATH)
